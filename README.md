# 🍔 Food Delivery App

### Developer: mohammed abed abed Alkareem abu khamash.
A modern Android application built with **Java** and **XML**, featuring a clean UI and object-oriented design.

## Features

- User registration and login
- Browse food products
- Search for food items
- View product details
- Add items to cart
- Add items to favorites
- Manage cart (increase/decrease quantity)
- Place orders
- View previous orders
- Separate data for each logged-in user
